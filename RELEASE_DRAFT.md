Description:

````markdown
Initial public release of mao-dialectics, an AI agent methodology skill inspired by dialectical analysis, contradiction analysis, practice theory, investigation, strategic planning, software architecture reasoning, brainstorming, project execution, and retrospection.

## Highlights

- OpenCode skill entrypoint through `SKILL.md`.
- Codex support through `SKILL.md`, `AGENTS.md`, and `agents/openai.yaml`.
- Claude Code support through `CLAUDE.md` and `/mao-dialectics` slash command.
- One-command local installer: `bash scripts/install.sh all`.
- Searchable docs for Codex, Claude Code, and OpenCode.
- Real examples for product strategy, coding architecture, retrospectives, and brainstorming.

## Install

```bash
git clone https://github.com/wangbh030722/mao-dialectics.git
cd mao-dialectics
bash scripts/install.sh all
```

## Disclaimer

This project is a methodology tool for AI agents. It is not political propaganda and is not intended for historical-person evaluation. Agent outputs should be verified in context.
````
