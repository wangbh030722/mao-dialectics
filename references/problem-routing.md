# 问题类型与分析方法路由

> **设计原则**：不同质的矛盾，用不同质的方法解决。一个问题属于什么类型，就用对应的毛选方法论框架去分析，而不是套一成不变的模板。
>
> 使用方式：当收到用户的提问或分析请求后，先用「问题类型诊断」判断属于哪一类，然后跳转到对应的分析方法。

---

## 快速诊断：问题属于哪种类型？

```
用户的问题/请求 →
                  ↓
  涉及方向/选择？    → 类型1: 战略决策型
  涉及竞品/各方？    → 类型2: 竞争格局型
  涉及内部管理？    → 类型3: 组织人事型
  涉及新事物？      → 类型4: 新生事物评估
  涉及信息不足？    → 类型5: 调查研究型
  涉及学习/认识？   → 类型6: 认识论型
  涉及危机/困境？   → 类型7: 危机困难型
  涉及方案/策略？   → 类型8: 政策方案制定
  涉及个人成长？    → 类型9: 自我修养型
  涉及合作/谈判？   → 类型10: 合作谈判型
  涉及判断大势？    → 类型11: 形势判断型
  涉及团队管理？    → 类型12: 领导管理型
  ---- 新增 ----
  涉及长期规划？    → 类型13: 战略规划型
  涉及创意/发散？   → 类型14: 头脑风暴型
  涉及做计划？      → 类型15: 项目计划型
  涉及复盘总结？    → 类型16: 复盘总结型
  不确定/综合      → 使用矛盾分析通用框架
```

## 类型1：战略决策型

**典型问题**：该往哪个方向走？A和B选哪个？现在该不该入场？

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **抓主要矛盾** | 《矛盾论》 | 在多个选项中找到决定性的那个 |
| **阶段论** | 《论持久战》 | 判断当前处于什么阶段，下一阶段是什么 |
| **矛盾转化分析** | 《矛盾论》 | 分析各选项的优缺点在什么条件下会转化 |
| **集中优势兵力** | 《集中优势兵力，各个歼灭敌人》 | 资源聚焦在最有希望的方向 |

### 分析框架

```
1. 当前的主要矛盾是什么？（不是所有问题都要解决）
2. 每个选项对应解决了哪个矛盾？
3. 各选项处于什么发展阶段？是萌芽、增长、成熟还是衰退？
4. 如果选A，优势在什么条件下会变成劣势？选B同理
5. 资源有限的情况下，应该「集中优势兵力」打哪个方向？
```

### 关键原文

> "集中优势兵力、各个歼灭敌人的原则，是我军从开始建军起十多年以来的优良传统。"（《集中优势兵力，各个歼灭敌人》）

> "事物发展的根本原因，不是在事物的外部而是在事物的内部，在于事物内部的矛盾性。"（《矛盾论》）

## 类型2：竞争格局型

**典型问题**：分析一下这个市场、这家公司怎么样、竞品对比

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **结构性分析** | 《中国社会各阶级的分析》 | 分析各方利益格局和立场 |
| **矛盾体系梳理** | 《矛盾论》 | 找出竞争中的主要矛盾和次要矛盾 |
| **强弱转化** | 《论持久战》 | 分析各方优劣势的转化趋势 |
| **纸老虎分析** | 《和美国记者安娜·路易斯·斯特朗的谈话》 | 区分表面强大和真正强大 |

### 分析框架

```
1. 格局中各方是谁？各自处于什么位置？
   - 谁在支配地位？谁在挑战？谁在跟随？
   - 各自的核心利益是什么？
2. 各方力量对比是静态的还是动态的？
   - 强中有弱、弱中有强——各自真正的优势和劣势是什么？
   - 哪些优势会随时间减弱？哪些劣势会随时间改善？
3. 中间力量是谁？
   - 它的两面性是什么？什么条件下会倒向哪边？
4. 本质上是实力竞争，还是路线/方向的竞争？
```

### 关键原文

> "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"（《中国社会各阶级的分析》）

> "一切反动派都是纸老虎。看起来，反动派的样子是可怕的，但是实际上并没有什么了不起的力量。"（《和美国记者安娜·路易斯·斯特朗的谈话》）

## 类型3：组织人事型

**典型问题**：团队不协调、内部矛盾处理、人员管理问题

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **批评与自我批评** | 《整顿党的作风》 | 处理组织内部矛盾的正确方式 |
| **反对自由主义** | 《反对自由主义》 | 识别组织内的纪律涣散问题 |
| **纠正错误思想** | 《关于纠正党内的错误思想》 | 识别和纠正组织中普遍的错误倾向 |
| **弹钢琴** | 《党委会的工作方法》 | 多任务协调与重点工作结合 |

### 分析框架

```
1. 组织的内部矛盾是什么性质？（对抗性还是非对抗性？）
2. 存在哪些自由主义表现？
   - 自由主义的11种表现中，哪些在当前组织中存在？
3. 组织中存在哪些错误思想倾向？
   - 单纯军事观点？极端民主化？绝对平均主义？个人主义？
4. 沟通方式是否健康？
   - 问题是否摆在桌面上？信息是否互通？
5. 是否在用「团结—批评—团结」的公式？
```

### 关键原文

> "我们主张积极的思想斗争，因为它是达到党内和革命团体内的团结使之利于战斗的武器。每个共产党员和革命分子，应该拿起这个武器。"（《反对自由主义》）

> "互相观望、互相等待、互相猜疑、互相戒备，这不是团结，这是分裂。"（《党委会的工作方法》中的互通情报原则）

## 类型4：新生事物评估

**典型问题**：这个新方向有前途吗？值不值得投入？一个新兴趋势的判断

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **生存条件分析** | 《中国的红色政权为什么能够存在》 | 新生事物在新环境下生存的必要条件 |
| **见微知著** | 《星星之火，可以燎原》 | 从早期迹象判断未来发展潜力 |
| **矛盾分析** | 《矛盾论》 | 新生事物代表了什么矛盾方向？ |
| **阶段判断** | 《中国革命战争的战略问题》 | 新生事物处于哪个发展阶段？ |

### 分析框架

```
1. 这个新生事物满足了什么需要？解决了什么矛盾？
2. 它的生存条件是否具备？
   - 客观条件（市场/技术/环境）是否成熟？
   - 主观力量（团队/资源/能力）是否匹配？
3. 它的方向是否符合历史趋势？（不是看它现在有多大，看它往哪走）
4. 目前的量变积累到什么程度了？距离质变的临界点还有多远？
5. 批评性自问：我是在客观判断，还是因为情绪（乐观/悲观）在做判断？
```

### 关键原文

> "它是站在海岸遥望海中已经看得见桅杆尖头了的一只航船，它是立于高山之巅远看东方已见光芒四射喷薄欲出的一轮朝日。"（《星星之火，可以燎原》）

> "一国之内，在四周白色政权的包围中，有一小块或若干小块红色政权的区域长期地存在，这是世界各国从来没有的事。这种奇事的发生，有其独特的原因。而其存在和发展，亦必有相当的条件。"（《中国的红色政权为什么能够存在》）

## 类型5：调查研究型

**典型问题**：不了解情况、信息不足、需要做判断但没有足够依据

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **没有调查就没有发言权** | 《反对本本主义》 | 调查的极端重要性 |
| **调查研究方法** | 《湖南农民运动考察报告》 | 如何做实地调查——14件大事的分类法 |
| **调查技术** | 《〈农村调查〉的序言和跋》 | 调查的具体方法和技术 |
| **实事求是** | 《改造我们的学习》 | 从事实中引出结论的方法 |

### 分析框架

```
1. 信息不足的判断，是应该暂缓结论、先去调查
2. 调查的方法：
   - 开调查会：找不同类型的人
   - 实地观察：亲自到现场
   - 阅读材料：注意来源可靠性
3. 收集信息时的原则：
   - 正反面兼看
   - 一手为主、二手为辅
   - 不要先有结论再找论据
4. 「去粗取精、去伪存真、由此及彼、由表及里」
```

### 关键原文

> "一切结论产生于调查情况的末尾，而不是在它的先头。"（《反对本本主义》）

> "你对于那个问题不能解决么？那末，你就去调查那个问题的现状和它的历史吧！你完完全全调查明白了，你对那个问题就有解决的办法了。"（《反对本本主义》）

## 类型6：认识论型

**典型问题**：理论与实践的关系、学习方向、如何建立正确的认知

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **实践认识循环** | 《实践论》 | 感性→理性→实践的认知规律 |
| **改造学习** | 《改造我们的学习》 | 反对主观主义的学习方法 |
| **反对党八股** | 《反对党八股》 | 防止形式主义和空洞表达 |
| **学风问题** | 《整顿党的作风》 | 反对理论和实际脱离 |

### 分析框架

```
1. 当前的问题是理论不足还是实践不足？
2. 如果是理论不足：
   - 是不是把局部经验当成了普遍真理？
   - 需要从实践中上升到理论的高度
3. 如果是实践不足：
   - 是不是只读书不看路？
   - 理论需要在具体条件下灵活运用
4. 检查是否有主观主义：
   - 是「从实际出发」还是「从本本出发」？
   - 有没有犯「左」或右的错误？
```

### 关键原文

> "实践、认识、再实践、再认识，这种形式，循环往复以至无穷，而实践和认识之每一循环的内容，都比较地进到了高一级的程度。"（《实践论》）

> "不注重研究现状，不注重研究历史，不注重马克思列宁主义的应用，这些都是极坏的作风。"（《改造我们的学习》）

## 类型7：危机困难型

**典型问题**：压力大、资源不足、形势不利、被压制

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **强弱转化** | 《论持久战》 | 不利局势中的转机分析 |
| **极端条件生存** | 《井冈山的斗争》 | 资源极度匮乏下的生存策略 |
| **战略上藐视，战术上重视** | 《和美国记者安娜·路易斯·斯特朗的谈话》 | 面对强大压力时的心态和方法 |
| **长期主义** | 《愚公移山》 | 坚持的力量 |
| **战略清醒** | 《丢掉幻想，准备斗争》 | 面对困难不抱幻想 |

### 分析框架

```
1. 这个困境的本质是什么？是暂时的还是根本性的？
2. 不利条件中，有没有同时存在的有利条件？（两面地看）
3. 对方的优势中有没有劣势？自己的劣势中有没有优势？
4. 在资源极度有限的情况下，应该「集中优势兵力」打哪个点？
5. 需要多长时间？是否有足够的韧性和耐心？
6. 有没有不切实际的幻想需要丢掉？
```

### 关键原文

> "我们面前的困难是有的，而且是很多的，但是我们确信：一切困难都将被全国人民的英勇奋斗所战胜。"（《论联合政府》）

> "有利的情况和主动的恢复，产生于'再坚持一下'的努力之中。"（《抗日游击战争的战略问题》）

## 类型8：政策方案制定

**典型问题**：怎么写方案、怎么制定策略、怎么定规矩

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **政策制定的辩证法** | 《论政策》 | 政策制定中的辩证思维 |
| **政策中的几个问题** | 《关于目前党的政策中的几个重要问题》 | 政策制定中要避免的常见错误 |
| **关心群众生活，注意工作方法** | 《关心群众生活，注意工作方法》 | 方案必须解决实际问题 |
| **集中起来、坚持下去** | 《关于领导方法的若干问题》 | 从群众中来到群众中去的方法 |

### 分析框架

```
1. 方案解决了什么矛盾？（没有明确要解决的问题，方案就没有方向）
2. 方案是否兼顾了各方利益？（特别是被忽略的群体的利益）
3. 方案是「从群众中来」的吗？还是坐在办公室拍脑袋的？
4. 方案在执行层面是否可行？
5. 是否考虑到了不同地区的不同情况（区别对待）？
6. 方案中有没有"左"或右的倾向？
```

### 关键原文

> "我们的原则是：『按照实际情况决定工作方针』，这是一切共产党员所必须牢牢记住的最基本的工作方法。"（《在晋绥干部会议上的讲话》）

> "一切空话都是无用的，必须给人民以看得见的物质福利。"（《关心群众生活，注意工作方法》）

## 类型9：自我修养型

**典型问题**：个人成长、职业规划、做人做事

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **纯粹精神** | 《纪念白求恩》 | 专业与奉献的精神 |
| **价值观方法论统一** | 《为人民服务》 | 做事的目的是什么 |
| **长期坚持** | 《愚公移山》 | 面对困难的态度 |
| **学习历史** | 《学习和时局》 | 从经验中学习 |
| **反对自由主义** | 《反对自由主义》 | 自我检查的镜子 |

### 分析框架

```
1. 做这件事的目的是什么？是为了谁？
2. 是否具备白求恩式的「极端负责任」精神？
3. 面对困难时，是「愚公移山」还是「知难而退」？
4. 有没有自由主义的表现？（对自己要求不严等）
5. 有没有从过去的经历中真正学到东西？
```

### 关键原文

> "一个人能力有大小，但只要有这点精神，就是一个高尚的人，一个纯粹的人，一个有道德的人，一个脱离了低级趣味的人，一个有益于人民的人。"（《纪念白求恩》）

> "人总是要死的，但死的意义有不同。……为人民利益而死，就比泰山还重。"（《为人民服务》）

## 类型10：合作谈判型

**典型问题**：要不要合作、怎么谈、团结谁反对谁、统一战线

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **统一战线的辩证** | 《统一战线中的独立自主问题》 | 合作中保持独立性的边界 |
| **策略分析** | 《目前抗日统一战线中的策略问题》 | 联合谁、中立谁、孤立谁 |
| **阶级分析** | 《中国社会各阶级的分析》 | 各方的利益和立场判断 |
| **有理有利有节** | 《目前抗日统一战线中的策略问题》 | 斗争中的分寸感 |

### 分析框架

```
1. 合作中对谁是主要矛盾？谁是次要矛盾？
2. 各方各自的利益是什么？底线在哪？
3. 合作中「独立自主」的边界在哪？什么可以让步，什么不能让？
4. 斗争策略：有理（道理上站得住）、有利（对自己有利）、有节（有节制不过火）
5. 对中间力量要「又联合又斗争」——不要只看一面
```

### 关键原文

> "我们的方针是统一战线中的独立自主，既统一，又独立。"（《统一战线中的独立自主问题》）

> "在抗日统一战线时期中，斗争是团结的手段，团结是斗争的目的。以斗争求团结则团结存，以退让求团结则团结亡。"（《目前抗日统一战线中的策略问题》）

## 类型11：形势判断型

**典型问题**：怎么看当前的大环境、怎么看趋势、大势如何

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **形势分析框架** | 《目前形势和我们的任务》 | 系统性的形势判断方法 |
| **战略问题方法论** | 《中国革命战争的战略问题》 | 战争/竞争的全局性分析 |
| **矛盾体系判断** | 《矛盾论》 | 主要矛盾的变化决定形势变化 |
| **转折点判断** | 《目前形势和我们的任务》 | 判断事物是否到了转折点 |

### 分析框架

```
1. 当前形势的本质特征是什么？（不是罗列现象，是找到本质）
2. 主要矛盾是什么？和之前比有没有变化？
3. 当前处在什么阶段？是量变积累还是质变临近？
4. 有利条件和不利条件分别是什么？各自的趋势如何？
5. 转折点是否临近？标志是什么？
```

### 关键原文

> "当形势改变的时候，我们的策略也必须随之改变。"（《目前形势和我们的任务》）

> "我们应当把世界进步的情况和黑暗的情况的基本区别，认清矛盾的实质，然后决定我们的态度。"（《关于目前国际形势的几点估计》）

## 类型12：领导管理型

**典型问题**：怎么带团队、怎么分配工作、怎么提高效率

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **弹钢琴** | 《党委会的工作方法》 | 多任务协同与重点工作结合 |
| **一般与个别相结合** | 《关于领导方法的若干问题》 | 从试点到推广的领导方法 |
| **领导与群众相结合** | 《关于领导方法的若干问题》 | 领导意图与群众意愿的统一 |
| **关心群众** | 《关心群众生活，注意工作方法》 | 工作要以解决实际问题为导向 |
| **组织起来** | 《组织起来》 | 组织动员的方法论 |

### 分析框架

```
1. 中心工作是什么？辅助工作是什么？（弹钢琴）
2. 有没有先试点再推广？（一般号召与个别指导相结合）
3. 领导和群众之间有没有信息断层？
4. 工作方法是否解决了群众关心的问题？
5. 队伍有没有被「组织起来」？还是是一盘散沙？
6. 领导干部有没有注意团结不同意见的人？
```

### 关键原文

> "弹钢琴要十个指头都动作，不能有的动，有的不动。但是，十个指头同时都按下去，那也不成调子。要产生好的音乐，十个指头的动作要有节奏，要互相配合。"（《党委会的工作方法》）

> "任何工作任务，如果没有一般的普遍的号召，就不能动员广大群众行动起来。但如果只限于一般号召，而领导人员没有具体地直接地从若干组织将所号召的工作深入实施，……就无法考验自己提出的一般号召是否正确。"（《关于领导方法的若干问题》）

## 类型13：战略规划型

**典型问题**：怎么定方向、做 roadmap、长期规划

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **战略防御→反攻** | 《中国革命战争的战略问题》 | 完整战略规划cycle |
| **阶段论** | 《论持久战》 | 分阶段规划 |
| **形势判断** | 《目前形势和我们的任务》 | 判断当前所处位置 |
| **战略预见** | 《星星之火，可以燎原》 | 从当前看到未来 |

### 规划框架

```
1. 形势分析：当前的全局情况是什么？
   - 有利条件和不利条件各是什么？
   - 主要矛盾是什么？它决定了什么方向是对的？
2. 阶段划分：整个过程可以分成几个阶段？
   - 当前阶段的核心任务是什么？
   - 进入下一阶段的条件是什么？
3. 方针制定：总的指导方针是什么？
   - 什么是"集中优势兵力要打的方向"？
   - 什么是应该放弃的？
4. 部署计划：方针如何转化为具体行动？
   - 谁在什么时间做什么？
   - 「抓而不紧，等于不抓」——计划是否可执行？
5. 风险预判：最坏的情况是什么？预案是什么？
   - 如果条件变化，计划如何调整？
```

### 关键原文

> "指挥员的正确的部署来源于正确的决心，正确的决心来源于正确的判断，正确的判断来源于周到的和必要的侦察，和对于各种侦察材料的联贯起来的思索。"（《中国革命战争的战略问题》）

## 类型14：头脑风暴型

**典型问题**：想方案、找灵感、创意发散、遇到瓶颈

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **百花齐放** | 《在延安文艺座谈会上的讲话》 | 鼓励多种可能性，不设限 |
| **普及与提高** | 《在延安文艺座谈会上的讲话》 | 先做出来再优化 |
| **反对党八股** | 《反对党八股》 | 反对空话套话，言之有物 |
| **去粗取精** | 《实践论》 | 从发散到收敛的筛选方法 |

### 创意框架

```
1. 问题定性：要解决的核心问题是什么？
   - 这是真问题还是伪问题？
   - 边界在哪里？什么是不需要解决的？
2. 百花齐放（发散阶段）：
   - 不设限制，输出各种可能性
   - 先不评判好坏，数量优先
   - 不同角度、不同层次的方案都列出来
3. 去粗取精（收敛阶段）：
   - 「去粗取精、去伪存真、由此及彼、由表及里」
   - 留下的选项是"真的好"还是"只是熟悉"？
4. 普及与提高（迭代阶段）：
   - 先做出一个可用的版本（普及）
   - 再迭代优化（提高）
   - 没有普及就没有提高的基础
```

### 关键原文

> "我们的提高，是在普及基础上的提高；我们的普及，是在提高指导下的普及。"（《在延安文艺座谈会上的讲话》）

## 类型16：复盘总结型

**典型问题**：做完回头看、总结经验、从失败中学习

### 核心方法

| 方法 | 来源 | 解决的问题 |
|------|------|-----------|
| **总结经验** | 《学习和时局》 | 从经历中提炼规律 |
| **批评与自我批评** | 《整顿党的作风》 | 正视错误，不回避 |
| **实践认识循环** | 《实践论》 | 从实践上升到理论 |
| **历史观点** | 《学习和时局》 | 把一件事放在更大的过程中看 |

### 复盘框架

```
1. 回顾事实
   - 原计划是什么？（目标、预期、方法）
   - 实际发生了什么？（结果、过程、关键事件）
   - 差距在哪？什么在预期内，什么出乎意料？
2. 分析原因
   - 成功/失败的根本原因是什么？
   - 主观原因和客观原因分别是什么？
   - 主要矛盾有没有发生变化？条件有没有变化？
3. 批评与自我批评
   - 哪些地方可以做得更好？
   - 不推卸、不回避、不归咎于他人
   - 「惩前毖后，治病救人」——批评的目的是进步
4. 提炼经验
   - 从这次经历中能抽象出什么可以复用的方法或原则？
   - 这是特殊情况，还是可以泛化的规律？
   - 如果再做一次，会有什么不同？
```

### 关键原文

> "所谓经验，就是实行政策的过程和归宿。……需要认真地总结经验，使经验带上条理性、综合性，上升成为理论。"（《学习和时局》）

> "犯了错误则要求改正，改正得越迅速，越彻底，越好。"（《整顿党的作风》）

## 综合型：通用矛盾分析框架

当问题无法明确归入以上类型，或涉及多个方面时，使用通用框架：

### 六步分析

| 步骤 | 内容 | 对应毛选方法论 |
|------|------|--------------|
| 1. 现象定位 | 区分现象与本质 | 《实践论》（感性→理性） |
| 2. 矛盾识别 | 找到主要矛盾和矛盾的主要方面 | 《矛盾论》 |
| 3. 历史溯源 | 追溯矛盾的产生和演变 | 《矛盾论》（矛盾的特殊性） |
| 4. 条件与趋势 | 分析转化条件和走向 | 《矛盾论》（同一性与斗争性） |
| 5. 批判性诊断 | 主流与支流、正面与反面 | 两点论 |
| 6. 实践建议 | 在当前条件下可操作的意见 | 《实践论》（理论→实践） |
