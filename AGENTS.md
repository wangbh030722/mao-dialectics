Use these references only when needed:

- `references/problem-routing.md` for choosing the right workflow by problem type.
- `references/contradiction.md` for contradiction analysis.
- `references/practice.md` for practice-based reasoning and retrospection.
- `references/methodology.md` for the broader methodology source map.
- `references/analytical-schema.md` for output schemas and examples.

Operational rules:

- Treat this as a methodology tool, not political propaganda.
- Do not make historical-person evaluation the center of the answer.
- Do not mechanically announce that you are using Mao methodology unless the user asks.
