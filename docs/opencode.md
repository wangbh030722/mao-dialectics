# OpenCode Skill Guide

`mao-dialectics` is directly usable as an OpenCode skill through `SKILL.md`.

## Install

```bash
git clone https://github.com/wangbh030722/mao-dialectics.git
cd mao-dialectics
bash scripts/install.sh opencode
```

The installer copies the skill to:

```text
~/.config/opencode/skills/mao-dialectics
```

## Use

```text
skill("mao-dialectics")
```

## Best-Fit Tasks

- dialectical analysis
- software architecture decisions
- strategy planning
- brainstorming
- project execution plans
- retrospectives
- investigation and research planning
